# Project-Management-System-PMS-
A console-based Project Management System developed using C++. The system allows users to: - Create and manage projects - Add, update, and assign tasks - Manage team members - Generate reports - Save and load data using file handling

## How to Run
1. Download the repository as ZIP
2. Extract files
3. Open `ProjectManagementSystem.cbp` in Code::Blocks
4. Click Build & Run

## Concepts Used
- Object-Oriented Programming
- STL (vector, map)
- File Handling
- Exception Handling
